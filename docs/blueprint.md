# **App Name**: HemoLink

## Core Features:

- Donor Registration & Profile: Allows donors to register, input personal details including blood group, city, last donation date, and availability, and manage their profile.
- Donor Search: Enables users (recipients, hospitals) to search for available donors by city and blood group.
- Blood Bank Locator: Locates nearby blood banks, displaying their address, contact information, and available blood types, integrated with Google Maps API.
- Emergency Request Management: Facilitates users to post urgent blood requests, specifying blood type, units required, and current location.
- Smart Notification System: Automatically sends SMS and email alerts to matching donors and nearby blood banks for emergency blood requests using Twilio and SendGrid APIs.
- User Authentication & Authorization: Provides secure login and registration functionalities for donors, recipients, and administrators, with role-based access control.
- Smart Donor Matching: An algorithm that intelligently matches blood requests with available donors based on location proximity, blood type compatibility, and availability.

## Style Guidelines:

- Primary color: Deep Red (#b30000) for visual prominence and association with blood and urgency.
- Background color: A very light, desaturated red-grey (#F5EEEE), maintaining a clean, minimal medical aesthetic while subtly referencing the primary color.
- Accent color: Dark Blue (#1A2F60), used for elements requiring emphasis, conveying trust and government reliability.
- Body and headline font: 'Inter', a grotesque-style sans-serif for its modern, neutral, and highly readable qualities, fitting a professional government application.
- Clean, minimal, and professional system-style outline icons to complement the overall government-ready design.
- Card-based layout with soft shadows for structured information, rounded buttons and input fields for a contemporary feel, and a fully mobile-responsive design for optimal viewing across devices.
- Smooth and subtle animations and transitions for a fluid and polished user experience, enhancing usability without distraction.
