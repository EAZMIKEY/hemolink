# HEMOLINK - Blood Donor Finder Application

HEMOLINK is a real-time blood donor connection platform built with **Next.js 15**, **Tailwind CSS**, and **Firebase**. It facilitates the connection between verified blood donors and those in need, helping save lives in India.

## Features

- **Connecting Donors**: Verified donor network with real-time status.
- **Search System**: Find donors by blood group and city.
- **Modern UI**: Built with Shadcn UI and Framer Motion for a premium experience.
- **Firebase Integration**: Secure backend powered by Firebase for data storage and authentication.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open your browser and navigate to `http://localhost:9002`.

## Project Structure

- `src/app/`: Next.js App Router for page definitions.
- `src/components/`: Reusable React components.
- `src/lib/`: Utility functions and helper scripts.
- `.agents/`: Antigravity workflows for project maintenance.
